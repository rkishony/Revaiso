# Revaiso – Feedback & Support

This is the public issue tracker for [Revaiso](https://workspace.google.com/marketplace), a Google Docs add-on that turns your editing instructions into native Google Docs suggestions.

## What is Revaiso?

Revaiso lets you describe how you want text revised, then applies AI output as Google Docs suggestions — so you review and approve every change.

- Select text and open Revaiso from the toolbar, right-click menu, or **Ctrl+Shift+K** (⌘⇧K on Mac)
- Describe your edit or ask it to insert new content
- Review changes using Google Docs' built-in Suggesting mode

## Report a Bug or Request a Feature

👉 [Open an issue](../../issues/new)

Please include:
- What you were trying to do
- What happened instead
- Your browser and OS

## FAQ

**Is my text sent to a server?**  
Yes — your selected text is sent to an AI model to generate suggestions. No data is stored beyond the request.

**Can I use my own API key?**  
Yes. Go to Revaiso Settings and enter your own key to use it instead of the hosted service.

**Which AI models are supported?**  
See Settings for the current list of supported models.
